# SWA — Databricks Assets

This repository holds Databricks notebooks and supporting material shared with Southwest
Airlines (SWA).

## Contents

| Folder | Description |
|---|---|
| [`Redshift Policy Sync`](./Redshift%20Policy%20Sync) | A Databricks notebook that extracts access-control privileges (GRANTs) from Amazon Redshift and reproduces them as equivalent Unity Catalog grants on a Lakehouse Federation foreign catalog. See the folder's [README](./Redshift%20Policy%20Sync/README.md) for full details. |

Each folder contains its own README with setup, usage, and configuration details.
